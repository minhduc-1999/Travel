## Cài đặt chương trình
#### Cài đặt packages

```bash 
  yarn
```
hoặc
```bash 
  npm install
```

#### Khởi động Metro

```bash 
  yarn start
```
hoặc
```bash 
  npx react-native start
```

#### Khởi động ứng dụng

```bash 
  yarn android
```
hoặc
```bash 
  npx react-native run-android
```
